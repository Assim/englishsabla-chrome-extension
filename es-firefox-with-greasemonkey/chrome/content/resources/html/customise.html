Activate Customisations

The English Sabla extension depends on an another extension called Greasemonkey to perform the customisation, please follow the steps below.

If you haven't done it, follow the steps below by clicking them:
1- Install Greasemonkey Addon (if you installed it, then skip to step 2)
2- Install Customisations to Greasemonkey (only works if step 1 completed)

If you have followed the steps above, then click the following button:
Customisation Options

Username:
Resize images in posts so they can fit the page
Hide forum categories
Hide signatures




If these settings are not working, then please re-activate the settings by clicking here.
